Create a Spring Boot Starter Project 
Add Spring Web,Eureka Server and Actuator Modules
Add the @EnableEurekaServer on the Java class with main method
Rename the application.properties as application.yml
Create Two Different Profile for configuring Eureka Server
Add the minimal Configuration properties required for each profile 
Run project as spring boot project 
Check URL on browser to see Eureka Server dashboard : http://localhost:8761/
